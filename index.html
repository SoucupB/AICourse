Examples

There are 2 problems which are mandatory for every machine learning engineer. This problems are the XOR problem where the structure should learn how to mimic the XOR binary function(The hello world for machine learning!) and the digits recognition problem which is training a neural Network to recognize handwritten digits.

Here is the code for digits training the digits recognition algorithm. 

                
import NeuralNetwork as nn #import the library
import FileReader as fr
images = fr.GetImageData() #get mnist data which are just the digits images formatted to binary
labels = fr.GetLabels() #get the labels for each image
imageHeight = 28
imageWidth = 28
epochs = 20
net = nn.NeuralNetwork([imageHeight * imageWidth, 51, 10], 0.2, [nn.RELU, nn.SIGMOID])
for i in range(20):
    print(i, net.sgdLong(images[:(50 * (i + 1))], labels[:(50 * (i + 1))], 10, nn.OPT_SGD))
